Un peu d'histoire

Le World Wide Web

  • Créé en 1989 par Tim Berners-Lee
  • Système hypertexte
  • Développé au CERN, maintenant géré par le W3C

Le tout premier navigateur

Le HTML

  • HyperText Markup Language
  • Permet de décrire le contenu d'une page Web
  • Tous les sites Web utilisent du HTML
  • HTML5 sorti en octobre 2014

Et concrètement ?

Squelette d'une page Web :


<!DOCTYPE html>
<html>
    <head>
        <title>Ma super page Web
    </head>
    <body>
        Hello world!
    </body>
</html>

Un peu plus loin


<!DOCTYPE html>
<html>
    <head>
        <title>Ma super page Web</title>
        <meta charset="utf-8">
        <link rel="stylesheet" href="design.css" />
        <script src="script.js"></script>
    </head>
    <body>
        <section>
            <h1>Ma section</h1>
            <article>
                <h2>Mon premier article</h2>
                <p>Hello world!</p>
            </article>
            <article>
                <h2>Mon second article</h2>
                <p>Hello world!²</p>
            </article>
        </section>
    </body>
</html>

Tada !
